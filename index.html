Linked List Cycle Detection

This project contains a python implementation of various cycle detection algorithms.

It also contains test code which creates lists with cycles (hence it doesn't use the default python list implementation), and graphs the efficiency of each algorithm when run against these lists.

Project Structure

  • LinkedList.py: Contains classes for linked lists and linked list nodes, and an implementation of the cycle detection algorithms.
  • GraphPlot.py: contains functions needed to plot the test data (as illustrated above). This calls the LinkedList class to generate lists with cycles, and runs a specified set of algorithms against each list.

Dependencies

  • Python 1.7
  • MatPlotLib, which requires NumPy
